# Task manager application

This app allows users to manage the tasks

#### Instructions
1.Download the source code
```bash
git clone [url to the github repo]
```
or just download the *.zip file

2.Install deps
```bash
npm install
```

3.Rn the app
```bash
node app
```

#### Deps list
- express
- pug
